# AI Game Gallery

Created for Claire Kim's workshop resources.

## Game Included

- Large Cave Treasure Escape

## Folder Structure

- `index.html` and `style.css`: gallery homepage
- `images/`: thumbnail images for the gallery
- `games/`: playable browser games
- Each game folder contains:
  - `index.html`
  - `style.css`
  - `script.js`
  - `assets/`

## Game Objective

The player starts at the cave entrance. They must explore one large cave with branching paths, collect the treasure deep inside, and return to the beginning to escape safely.

## How to Use

Open `index.html` in a browser to view the gallery.

To upload to GitHub Pages, upload the entire `ai-game-gallery` folder contents into a GitHub repository, then enable GitHub Pages from the repository settings.
